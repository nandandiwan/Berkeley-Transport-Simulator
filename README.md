# Silicon Nano-Transistor NEGF Simulation

This project is an atomistic simulation for nanotransistor and other nanodevices. Solves tight-binding NEGF with Poisson's equation self consistently (with FENICS finite element) to produce relevant IV curves and electronic properties.  Currently a work in progress. Basic simulations with NEGF/Poisson and past drift diffusion simulations in old_sims folder.
